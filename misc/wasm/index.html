    Try it locally (no terminal expertise needed)

    1. Install Go 1.22+ from go.dev/dl/.
    2. Open a terminal in this misc/wasm folder (or right-click “Open in Terminal”).
    3. Run this single command:
      GOOS=js GOARCH=wasm go build -o main.wasm main.go && go run web/web.go
    4. Your browser opens at http://localhost:8080. Add and check off TODOs!

    The server just serves this folder; nothing else is required.
